# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1, as its Code of Conduct.

In short: be respectful and considerate. We are committed to a welcoming, friendly experience
for everyone participating in the project, regardless of background or identity.

## Scope

This applies to all project spaces — issues, pull requests, discussions, and any official
communication channel — and when an individual is representing the project in public spaces.

## Reporting

To report a concern, contact the maintainer [@showjihyun](https://github.com/showjihyun).
All reports will be reviewed and handled confidentially.

The full text and enforcement guidelines are available at the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
